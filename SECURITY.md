## Reporting a Vulnerability

Did you find an security issue?
Please keep it private and submit a report [here](https://github.com/QKing-Official/LENAv2/security/advisories/new).
I will review it and patch it as soon as I can. To keep abusers from using the vulnerability please keep this private.
